Plateforme d’Apprentissage et d’Enseignement de Compétences
1. Introduction
1.1 Objectif
Ce document décrit les fonctionnalités de la plateforme.

La plateforme permet aux utilisateurs d’apprendre OU d’enseigner des compétences, de communiquer via chat, de gagner des points (XP) et d’obtenir des certificats.


1.2 Portée
Le système permet :

Un compte utilisateur avec un seul rôle (Apprenant OU Formateur)
Des demandes d’apprentissage
Des sessions (chat)
Un système de points (XP, niveaux, badges)
Des certificats
Un espace administrateur

1.3 Définitions
Utilisateur : personne inscrite
Apprenant : personne qui apprend
Formateur : personne qui enseigne
Session : échange via chat
XP : points d’expérience
Admin : administrateur

2. Acteurs
2.1 Apprenant
Peut :

Apprendre des compétences
Rejoindre des sessions
Laisser des avis
Gagner des XP

2.2 Formateur
Peut :

Enseigner des compétences
Accepter/refuser des demandes
Faire des sessions
Gagner des XP et badges
Demander un certificat

2.3 Administrateur
Peut :

Gérer les utilisateurs et compétences
Valider les certificats
Surveiller la plateforme

3. Vue du Système
Application web composée de :

Authentification et profils
Système d’apprentissage et d’enseignement
Chat en temps réel
Système de gamification
Tableau de bord admin
